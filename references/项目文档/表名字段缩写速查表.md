# 表名字段缩写速查表

这份表给零基础新人使用。

目的不是一次性背完所有缩写，而是遇到表名、字段名时，先能把它翻译成人话，避免“会看 Go 但看不懂业务命名”。

## 使用规则

以后看到缩写字段，尽量主动按下面格式解释：

- 字段名
- 拼音展开
- 中文含义
- 当前模块里的作用

例如：

- `rzzt`
  - 拼音：`ren zheng zhuang tai`
  - 中文：认证状态
  - 作用：表示发票当前是否已认证

## 一、通用上下文类

- `org`
  - 通常指机构

- `com`
  - 通常指公司、企业

- `comId`
  - 企业 ID

- `orgId`
  - 机构 ID

- `period`
  - 账期

- `reqNo`
  - 请求号、任务请求流水号

- `status`
  - 状态

- `remark`
  - 备注

## 四、税务与任务常见缩写

- `sb`
  - 拼音：`shen bao`
  - 中文：申报

- `kk`
  - 拼音：`kou kuan`
  - 中文：扣款

- `sk`
  - 拼音：`shui kuan`
  - 中文：税款

- `gs`
  - 在不同上下文里可能不同
  - 在税务模块里经常指税表主对象相关前缀，阅读时要结合上下文判断

- `gs_main`
  - 税表主记录

- `tax_task_main`
  - 税务任务主记录

## 六、阅读建议

- 不要求第一次就记住全部缩写
- 但在新手模式下，GPT 应该主动翻译缩写，不要等用户来问
- 如果某个缩写在不同上下文含义不同，要明确说“这里按当前模块语义解释”
